A good alternative would have 1. Actual access points that are affordable 2. A controller that can be self hosted (Preferably on something lightweight like a raspberry pi) 3. The ability to have multiple access points all managed by the controller from #2 as well as be PoE compatible 4. Basic functionality including optional client isolation (per ssid or vlan), vlan support, and multiple SSIDs connected each to different vlans 5. No recurring fees, whatsoever. As long as it meets those, I'm open to any potential options Bonus features would include captive portal support and site survey options, but these are 100% optional in my use cases.
