g/technology: NVCleanstall lets you customize the NVIDIA GeForce Driver package by removing components that you don't need (or want). This not only keeps things tidy, but also lowers disk usage and memory footprint. Unlike the NVIDIA custom installation, you can unselect packages like "Stereo 3D Glasses", "USB-C", Notebook Optimizations or Telemetry from the NVIDIA drivers.
