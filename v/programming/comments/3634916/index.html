Python OSINT tool for pentesters

6    31 Jan 2020 12:31 by u/latest-release

This OSINT tool is theHarvester which has the following features

  • Subdomain finder
  • Port scanning
  • Names
  • Virtualhost
import theHarvester
from theHarvester.discovery import baidusearch
from theHarvester.discovery import bingsearch
from theHarvester.discovery import dnsdumpster
from theHarvester.discovery import googlesearch

That's how easy it is to use this tool after installing it

pip3 install theHarvester

This tool requires python3.7 and thats how we did intergrate this tool at theharvester online
