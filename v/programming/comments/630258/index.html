COGTFO Code of Conduct Proposal

19    02 Nov 2015 02:11 by u/gtcom

The COGTFO Code of Conduct Proposal has the following points to be considered before being adopted by your programming community:

  1. If a community member's submission to the project includes one of the following, it will be considered acceptable:
    • Actual patches
    • Bug reports
    • Program feature requests
  2. If your submission does not fall under one of the above categories then, get the fuck out.
  3. There is no point 3.

The COGTFO Code of Conduct Proposal is covered under the WTFPL.
