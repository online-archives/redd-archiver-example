Checking username availability for voat.co using bash and curl

7    11 Jul 2015 07:43 by u/due

Here is the script for checking username availability for voat.co. You'll need to change the directory that contains the words.txt file.

~/projects/misc/voat_username_avialability/index.sh

IFS=$'\n' read -d '' -r -a lines < ~/projects/misc/voat_username_avialability/words.txt
for x in "${lines[@]}"
do
  echo "${x}\n"
  curl --data "userName=${x}" https://voat.co/account/CheckUsernameAvailability
  sleep 2s
  echo "\n"
done
