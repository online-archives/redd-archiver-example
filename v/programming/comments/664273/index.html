New RSS web app please-feed.me

7    17 Nov 2015 18:26 by u/real_billmo

Hey guys,

I made a simple web app using reddits API that you can add in interests that culminate into a large feed. Some features include infinite scroll, pull down from the top to refresh, as well as an easy filter tab so that you can specify an interest you want to see. It's made purely for consumption and has no social features but has no lack of content considering it's all coming from reddit. Also note that it's made primarily for phones (swipe gestures and home screen icons) but works in browsers too.

It is in beta mode and I'm continuing to work on it, but just thought someone may like it and want to use it. It's fun and free.

http://www.please-feed.me

Hopefully you all like it.
