Seeking Advice for a Stupid Fan Game

4    22 Jun 2015 07:02 by u/CaptainTickles

Right now I have some Python experience (mostly simple desktop GUI apps) and Java experience (simple desktop Swing-based apps and some server JSP stuff). I'd like to make a really stupid Five Nights at Freddy's fan game, and explore the world of simple indie game design in general.

I'd like to do this because I love to tell stories, and I'd like to tell some silly little stories through the medium of games. I'd like to start with a simple Five Nights at Freddy's game both because I would find it humorous to do, and as a way to start learning about game design.

I could expand on my Python knowledge and learn Pygame, or I could expand on my Java knowledge and learn... whatever Java game engine would be appropriate. Or I could do something else.

Dear, gentle people, what would be your advice for a direction to start in?
