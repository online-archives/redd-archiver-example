The difference between success and failure of any project comes to one word: "Why"

This clip portrays it perfectly "Race to Save River Song - Forest of the Dead - Doctor Who - BBC: https://www.youtube.com/watch?v=hzeeAUeIOzc"

It is Doctor Who, but that is irrelevant. Also the story is irrelevant. What is relevant is that you use the word "Why"... passionately.

  • Why does this code work? Why?
  • Someone created this code like this, But Why?
  • This code has worked for so many years. Why does it fail now? Why not before? Why?
  • Users have reported it but I can't reproduce? Why? Why them but I can't not me? Why?
