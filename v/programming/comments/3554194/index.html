Rust Actix Has Won The Internets!

1    08 Dec 2019 21:12 by u/libman

Rust Actix is the only web framework to finish in Top 10 of all TechEmpower benchmarks.

This is true in the last published "round" (#18) as well as the latest daily tests.

Actix comes in at #1 of 396 frameworks in the latest Fortunes benchmark (the flagship), #9 in the JSON test, #2 in Single Query, #3 in Multiple Queries, #2 in Data Updates, and #7 in Plain Text.

Select C/C++ and Java frameworks beat it some of the time, but nothing beats it consistently. The Java/.NET frameworks that come close also use a ton more memory, which makes them a lot more expensive to deploy (and easier to DDoS).

Anyone wishing to write scalable Web APIs would be very foolish to use scriptkiddie frameworks like Django and Rails.
